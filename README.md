## 專案簡介
這個專案使用Unity製作，核心特色是把戰鬥中的行為都抽象成 `GameAction`，由 `ActionSystem` 統一使用。
目前已具備可遊玩的完整流程：`開始遊戲` ➔ `地圖選點` ➔ `戰鬥/商店/休息` ➔ `獲勝獎勵` ➔ `繼續推圖` ➔ `結束畫面`。

## 核心玩法
* 玩家每回合用三點法力打出手牌。
* 敵人有意圖系統，玩家可預判敵方行動。
* 戰鬥獲勝可選卡，逐步構築牌組。
* 地圖有路線選擇。

##  目前已實作內容
* `ActionSystem` 三階段反應：PRE / PER / POST
* **卡牌系統：** 抽牌、出牌、棄牌、洗牌
* **目標系統：** 單體手動指定、隨機目標、全體目標、自身目標
* **敵方機制：** 敵人意圖UI與回合執行
* **狀態效果：** Armor / Burn / Weak / Vulnerable
* **商店系統：** 消費購卡
* **休息節點：** 回血或是升級
* **戰鬥後卡牌獎勵：** 三選一卡牌
* **地圖進度：** 地圖節點與推進
* **介面體驗：** 手牌扇形排版、懸停放大、拖曳出牌

## 系統設計重點

* **以 `GameAction` 為中心：**
  所有戰鬥行為（傷害、抽牌、擊殺、狀態）都轉成Action進序列。
* **以資料驅動為主：**
  卡牌、敵人、英雄都由 `ScriptableObject` 配置，方便擴充與平衡。
